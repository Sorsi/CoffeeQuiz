# CoffeeQuiz
Trivia quiz app

# First step

Clone this repo with:
### `git clone https://github.com/Sorsi/CoffeeQuiz.git`

# Second step

Go to the folder:
### 'cd CoffeeQuiz/coffe-quiz-app'

# Third step

Install dependencies with:
### `npm i`

# Last step

Run it with:
### `npm start`

Easy, right?
